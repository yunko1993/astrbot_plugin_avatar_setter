# 🛡️ DNF 角色图展示助手 (AstrBot 插件)

🎭 **专为 DNF 群聊设计**：发图并 @某人 加上关键词 `全家福`，即可自动设置该群友的角色截图。之后任何人 @他 并带上 `全家福` 即可查看。

## ✨ 功能亮点
- 🧠 **智能识别**：无需输入任何 `/` 开头的指令，直接发送“图片 + @人 + 全家福”即可设置。
- 🔍 **精准触发**：必须包含关键词 `全家福` 才会响应，日常聊天中普通的 `@人` 绝不误触发。
- 🔒 **权限隔离**：
  - **普通用户**：只能设置/覆盖**自己**的角色图。
  - **管理员**：可以设置或修改**任何群友**的角色图。
- 🌐 **异步下载**：采用 `httpx` 高性能异步下载，图片保存不阻塞机器人。
- 📂 **持久化存储**：图片和配置保存在 `data/plugin_data/` 下，更新插件或重装代码，数据永不丢失。

## 📦 安装方法

### 1. 环境准备
本插件需要 `httpx` 库，请在服务器执行：
```bash
pip install httpx
```

### 2. 插件安装
- **WebUI 安装**：在 AstrBot 插件市场输入本仓库 GitHub 链接。
- **手动安装**：将代码放入 `data/plugins/astrbot_plugin_avatar_setter`。

## 🎮 使用指南

### 1. 设置角色图 (附带图片)
| 操作角色 | 动作 | 示例 |
| :--- | :--- | :--- |
| **普通用户** | 发送截图 + @自己 + 全家福 | `[图片] @小青菜 全家福` |
| **管理员** | 发送截图 + @任何人 + 全家福 | `[图片] @马化腾 全家福` |

### 2. 查看角色图 (仅限文字)
| 动作 | 示例 | 效果 |
| :--- | :--- | :--- |
| **任何人** | @目标群友 + 全家福 | 机器人直接回复该群友的截图 |

### 3. 管理指令
| 指令 | 说明 | 权限要求 |
| :--- | :--- | :--- |
| `/add_avatar_admin QQ号` | 添加一名新的插件管理员 | 仅限管理员 |

## ⚙️ 配置文件
插件自动在 `data/plugin_data/astrbot_plugin_avatar_setter/` 生成 `config.json`：

```json
{
    "admin_qq": ["1023902556"], 
    "avatars": {
        "1023902556": "1023902556.jpg"
    }
}
```

## 📂 目录结构
```text
AstrBot/
└── data/
    └── plugin_data/
        └── astrbot_plugin_avatar_setter/
            ├── config.json          # 数据库
            └── avatars/             # 存放所有角色截图 (.jpg)
```
