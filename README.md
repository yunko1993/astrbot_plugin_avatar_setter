# 🛡️ DNF 角色图展示助手 (AstrBot 插件)

🎭 **专为 DNF 群聊设计**：群友发送 `@某人 全家福` 即可自动获取或设置该群友的角色展示图（全家福截图）。

## ✨ 功能亮点
- 🔍 **精准触发**：必须包含关键词 `全家福` 才会响应，日常聊天中普通的 `@人` 不会触发，杜绝刷屏。
- 🔒 **权限隔离**：
  - **普通用户**：只能设置自己的角色图（发送图片 + `@自己 全家福`）。
  - **管理员**：可以设置或修改任何人的角色图。
- 📂 **数据规范**：所有配置与图片自动存储于 AstrBot 标准数据目录 `data/plugin_data/astrbot_plugin_avatar_setter/`，重装或更新插件配置不丢失。
- 🚀 **零配置**：安装即用，首次运行自动生成所有必要的文件夹和配置文件。

## 📦 安装方法

### 方式一：通过 AstrBot WebUI 安装 (推荐)
1. 打开 AstrBot 仪表盘 -> **插件市场**。
2. 点击 **通过 GitHub URL 安装**。
3. 输入本仓库链接：`https://github.com/你的用户名/astrbot_plugin_avatar_setter`
4. 点击安装并启用。

### 方式二：手动安装
1. 将本仓库下载并解压到 `data/plugins/astrbot_plugin_avatar_setter`。
2. 重启 AstrBot 即可。

## 🎮 使用指南

### 1. 设置/更换角色图
- **普通用户**：发送一张角色截图，并在附言中输入 `@自己 全家福`。
- **管理员**：发送一张角色截图，并在附言中输入 `@目标群友 全家福`。

### 2. 查看角色图
- **任何人**：直接发送文本 `@目标群友 全家福` 即可。机器人会自动回复该群友预设的截图。

### ❌ 错误反馈参考
| 操作场景 | 机器人反应 | 原因 |
| :--- | :--- | :--- |
| 只 @张三 不带关键词 | (无反应) | 缺少关键词 `全家福` |
| 普通用户 @张三 全家福 | ❌ 权限不足 | 只有管理员能给别人设图 |
| @自己 全家福 (不发图) | ⚠️ 请附上图片 | 未检测到图片消息段 |

## ⚙️ 配置文件说明
插件会在 `data/plugin_data/astrbot_plugin_avatar_setter/` 目录下生成 `config.json`：

```json
{
    "admin_qq": ["1023902556"], 
    "avatars": {
        "12345678": "12345678.jpg",
        "1023902556": "1023902556.png"
    }
}
```
> **提示**：如需新增管理员，手动修改 `admin_qq` 列表中的 QQ 号并重启机器人即可。

## 📂 目录结构
```text
AstrBot/
└── data/
    └── plugin_data/
        └── astrbot_plugin_avatar_setter/
            ├── config.json          # 记录 QQ 与图片的对应关系
            └── avatars/             # 存放所有群友上传的角色图
                ├── 1023902556.jpg
                └── 88888888.png
```
